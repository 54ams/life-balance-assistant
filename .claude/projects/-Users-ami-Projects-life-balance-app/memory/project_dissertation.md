- BCS synoptic project exploring how wearable data, emotional self-report, and lifestyle signals combine into a transparent, explainable Life Balance Index (LBI)
- Study: feasibility study with 5-8 participants over 7-14 days
- Hypotheses H1-H7 (all exploratory), H3 = adherence → next-day LBI
- Primary outcome: SUS usability score
- Backend on Render (life-balance-backend), web app also on Render (life-balance-web)
- Mobile builds via EAS (iOS + Android preview profiles)
- Stack: React Native + Expo Router, Node.js backend, WHOOP OAuth, OpenAI for reflections
- Demo mode available (30-day synthetic data, no WHOOP needed)

**Why:** Deadline-driven, aiming for very high first class. Every feature must be justified and traceable to the dissertation.

**How to apply:** Prioritise features that directly support dissertation claims. Every UI decision should be defensible in the write-up.
